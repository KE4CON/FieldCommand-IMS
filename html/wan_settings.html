USB Backup Drive

Any USB drive formatted and labelled FIELDCOMMAND (all caps) triggers automatic backup when plugged in. Works with any brand — LaCie, WD, SanDisk, Seagate, or generic — any size, ext4 or exFAT format.
To change the trigger label, also update /etc/udev/rules.d/99-fieldcommand-backup.rules and run sudo udevadm control --reload-rules.

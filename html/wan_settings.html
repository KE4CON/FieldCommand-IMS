ADMIN WAN / Internet Settings
Configure internet sources displayed on dashboard and wan-status page
← Dashboard
Settings are saved to /opt/fieldcommand/data/wan_config.json on the Pi and take effect within 30 seconds (next poll cycle). Any USB drive labelled FIELDCOMMAND triggers an automatic backup — works with any brand or size.

Primary Internet Source

Detection: when internet is up, if the modem admin page (or ping host) is reachable, this source is declared active. Set this to your modem/router LAN admin IP. For InstyConnect: http://10.1.1.1. For most home routers: http://192.168.1.1.

Secondary Internet Source

Common ping hosts: Starlink 192.168.100.1 · HughesNet 192.168.0.1 · ViaSat 192.168.100.1 · OneWeb varies by terminal model. The host must be reachable only when this connection is active.

USB Backup Drive

Any USB drive formatted and labelled FIELDCOMMAND (all caps) will trigger automatic backup when plugged in. Works with any brand — LaCie, WD, SanDisk, Seagate, generic — any size, ext4 or exFAT format.
To rename the trigger label, you must also update the udev rule at /etc/udev/rules.d/99-fieldcommand-backup.rules on the Pi and reload udev. The display name is cosmetic only.
